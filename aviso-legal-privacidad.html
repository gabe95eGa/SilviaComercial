2. Objeto de la web y condiciones de uso

La web ofrece información comercial sobre productos profesionales de peluquería y estética, datos de contacto del comercio y un formulario de prepedido para preparar productos antes de la recogida en tienda.

El prepedido no implica pago online. El cliente abonará el pedido en persona en la tienda el día de recogida indicado en el formulario o acordado con el comercio.

El usuario se compromete a facilitar datos veraces, exactos y actualizados cuando utilice el formulario. Sílvia Comercial podrá contactar con el usuario para confirmar disponibilidad, resolver dudas o ajustar el prepedido antes de la recogida.

Los precios, productos y disponibilidad pueden variar. La confirmación final del producto, precio y recogida se realiza directamente con el comercio.

3. Política de privacidad

Responsable del tratamiento

El responsable del tratamiento es Silvia Aparicio Xarau, titular de Sílvia Comercial de Perruqueria i Estètica.

Datos tratados

Se tratan los datos que el usuario facilita voluntariamente al preparar un prepedido o contactar con el comercio: nombre, teléfono si se indica, fecha de recogida, productos solicitados y notas adicionales.

No se solicitan datos especialmente protegidos. Si el usuario incluye información sensible en el campo de notas, lo hace bajo su responsabilidad y se recomienda no introducir datos que no sean necesarios para gestionar el prepedido.

Finalidades

  • Preparar y gestionar prepedidos para recogida en tienda.
  • Responder consultas realizadas por teléfono, WhatsApp, Instagram u otros canales indicados en la web.
  • Confirmar disponibilidad, aclarar incidencias o coordinar la recogida.
  • Cumplir obligaciones legales cuando proceda.

Legitimación

La base jurídica principal es la solicitud del usuario y su consentimiento al enviar voluntariamente el mensaje de prepedido o contactar con el comercio. En su caso, también puede existir interés legítimo para responder a consultas o gestionar comunicaciones vinculadas a una solicitud previa.

Conservación

Los datos se conservarán durante el tiempo necesario para gestionar la solicitud, atender posibles incidencias y cumplir las obligaciones legales aplicables. Los borradores guardados en este dispositivo permanecen en el propio navegador del usuario hasta que se borren o se limpien los datos del navegador.

Destinatarios

No se cederán datos a terceros salvo obligación legal o cuando sea necesario para gestionar la comunicación solicitada por el usuario. Al enviar el prepedido por WhatsApp, el mensaje se tramita a través de la plataforma WhatsApp, cuyas condiciones y políticas son ajenas a Sílvia Comercial.

Derechos

El usuario puede solicitar el acceso, rectificación, supresión, oposición, limitación del tratamiento y portabilidad de sus datos cuando proceda, contactando con el comercio mediante los teléfonos indicados. También puede presentar una reclamación ante la Agencia Española de Protección de Datos si considera que sus derechos no han sido atendidos correctamente.

4. Cookies y almacenamiento local

Esta web no utiliza cookies publicitarias ni analíticas. Utiliza almacenamiento local técnico del navegador para guardar el borrador del prepedido en el propio dispositivo del usuario, con la finalidad de no perder la información introducida al refrescar la página.

El usuario puede borrar este almacenamiento limpiando los datos del navegador o eliminando el borrador desde el propio dispositivo. Si en el futuro se incorporan herramientas analíticas, publicitarias o de terceros que requieran consentimiento, se actualizará esta información y se habilitará el mecanismo correspondiente.

5. Redes sociales y enlaces externos

La web puede enlazar a perfiles sociales del comercio, como Instagram, WhatsApp u otros canales que puedan variar con el tiempo. Al acceder a estas plataformas, el usuario queda sujeto a las condiciones y políticas de privacidad de cada servicio externo.

Sílvia Comercial no se hace responsable del contenido, disponibilidad o tratamiento de datos realizado por sitios web o plataformas de terceros enlazadas desde esta web.

6. Seguridad y actualización de la información

Sílvia Comercial aplicará medidas razonables para tratar la información con confidencialidad y usarla solo para las finalidades indicadas. No obstante, ningún sistema conectado a Internet puede garantizar una seguridad absoluta.

Esta información podrá actualizarse cuando cambien los servicios, canales de contacto, redes sociales, normativa aplicable o forma de gestionar los prepedidos.
